TEQneers Search Engine Optimization

This extension provides an indexed google/xml-sitemap, enhanced metatag-support and pagetitel-manipulations for TYPO3.
It's an replacement of the "metatag"-extension.

Support:    http://forge.typo3.org/projects/extension-tq_seo/
Bugtracker: http://forge.typo3.org/projects/extension-tq_seo/issues
Homepage:   http://opensource.teqneers.com/typo3/tq-seo.html
